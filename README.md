# Nexa Wallet

A rich, native Android wallet app to store your Nexa's in one place.


## Features

- Clean Dashboard
- Create & Import BIP39 Wallet
- Buy, Send, Receive NEXA
- View Transactions
- Light/dark mode
- QR Code Scan & Read
